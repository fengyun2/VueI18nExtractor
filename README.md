## 目录

```text
/vue-i18n-extractor
 node_modules/               # Node模块文件夹
 .gitignore                  # Git忽略文件
 package.json                # 项目配置文件
 package-lock.json           # 锁定安装版本文件

 output/                     # 输出目录
```
